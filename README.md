# Agentmail Slack Drafter

Lives in Slack and turns conversation content into outbound emails sent via [AgentMail](https://agentmail.to). The agent drafts every email in-thread and waits for explicit human approval before sending — nothing goes out without a review.

## Prerequisites
- An [AgentMail](https://agentmail.to) account with an API key
- A Slack workspace you can connect via Valet's Slack OAuth flow
